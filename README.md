# COMP2090SEF-Project-GPR_72

## Project Overview
This repository contains the two-part project for the Data Structures, Algorithms and Problem Solving course. The project integrates Object-Oriented Programming (OOP) with the study of advanced data structures and algorithms.

* **Part 1: Healthcare Tracking System (OOP Application)**: A Python-based interactive desktop application that utilizes core OOP concepts including classes, encapsulation, and modular programming to manage personal health metrics.
* **Part 2: Advanced Maze Generation (Self-Study)**: A technical implementation and study of Disjoint Set Union (DSU) and Randomized Kruskal’s Algorithm to generate perfect mazes.

---

## 📂 Repository Layout
The project is organized into the following directory structure to ensure clarity and meet submission requirements:

```text

├── Part 1/                  # Task 1: OOP-based Healthcare Application
│   ├── determine/           # Module for vitals interpretation logic

│   ├── Base.py              # User profile and data persistence management
│   ├── GUI.py              # Main application entry point and UI logic    <------- main.py
│   ├── PlotGraph.py         # Matplotlib chart generation logic
│   ├── ReportGenerator.py   # PDF report generation logic
│   ├── exercise_dataset.csv # Calorie calculation reference data
│   ├── question.py          # Question sets for mental health assessments
│   ├── user_health_data.csv # Historical record logs
│   ├── user_profile.json    # Basic user metrics storage
│   └── README.md            # Detailed documentation for Task 1      <----- README for task 1
│
├── Part 2/                  # Task 2: Self-study on Data Structures & Algorithms
│   ├── main.py              # Maze game entry point and Pygame loop      <------ main.py
│   ├── maze.py              # DSU implementation and Kruskal's algorithm
│   └── README.md            # Detailed documentation for Task 2      <----- README for task 2
│

```
